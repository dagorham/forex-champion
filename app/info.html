Currency Exchange Data Visualization

What is this?

This is a basic web application that allows you to plot the exchange rates of various currencies over time. It has support for historical as well as recent daily data. The daily data provides the exchange rate each minute over the past 24 hours. The historical data gives a variety of daily metrics over a user defined period of time. All exchange rates are in USD. The following currencies are supported:

  • Euro (European Union)
  • Pound (Great Britain)
  • Peso (Mexico)
  • Yen (Japan)
  • Dollar (Australia)
  • Dollar (Canada)
  • Rupee (India)
  • Yuan (China)
  • Riyal (Saudi Arabia)
  • Rand (South Africa)
  • Gold

How does it work?

For both historical and daily data, you may plot up to two currencies at any given time. Select the first currency and the second currency using the first two dropdown menus. If you only want to plot one currency, just leave the second menu as "None". Next, choose whether you want daily historical data or just the past 24 hours using the "Data Range" drop down menu.

For 24 hour data, these are the only options. For historical data, you will then need to specify a date range. The available data ranges from 2007/10/05 to yesterday's date. The dates must be input in one of the following formats:

  • YYYY/MM/DD
  • YYYY-MM-DD
  • YYYY MM DD

If a date box is left blank, or it is entered incorrectly, the date will automatically revert to the minimum or maximum possible value.

The final option for historical data is the data you would like to see. The following options are available:

  • Open Bid/Ask - Price at midnight UTC
  • Maximum Bid/Ask - Maximum price over the course of the day
  • Minimum Bid/Ask - Minimum price over the course of the day
  • Close Bid/Ask - Price at 11:59 PM UTC
